# Wordpress小技巧

收集wordpress的使用技巧。
## 在文章内容里插入Short Code

写关于插件的教程时可能需要插入short code，而wordpress会自动对文章内容里的short code进行解释，从而导致无法正常显示short code，要达到以上目的只需要给short code多添加一层中括号即可。如下所示:

	
	[[short code]]

会实际显示为

	
	[short code]
