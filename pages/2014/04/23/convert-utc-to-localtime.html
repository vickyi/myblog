如何将UTC时间转成 local datetime

Date 2014-04-23 17:59:11 Tags Python / UTC / datetime

在Python项目中，通常我们会把时间按UTC格式来存储，在显示时用本地的方式来显示，现在的问题是，如何将UTC时间转成 local datetime呢？

下面提供了一种方法, 将当前的UTC时间转成上海(中国大陆时区)时间：

>>> import pytz
>>> import datetime
>>> local_tz = pytz.timezone('Asia/Shanghai')
>>> def utc_2_loctime(utc_dt):
...     local_dt = utc_dt.replace(tzinfo=pytz.utc).astimezone(local_tz)
...     return local_tz.normalize(local_dt)
...
>>> print utc_2_loctime(datetime.datetime.utcnow())
2014-04-23 17:50:58.711877+08:00
>>> print datetime.datetime.utcnow()
2014-04-23 09:51:12.381790

这样就可以了。

Python里处理UTC的办法很多，还有一些第三方库比如 dateutil。
